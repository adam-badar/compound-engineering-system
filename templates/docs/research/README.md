# Research Artifacts

Use this folder for deep research outputs created by:

- `/compound-engineering-core:workflows:research`
- `/compound-engineering-core:workflows:deepen-plan`

## Naming

- Standalone research: `YYYY-MM-DD-<topic>-research.md`
- Plan deepening evidence belongs in `docs/reviews/plans/` as `<plan-slug>-deepen-research.md`

## Minimum Expectations

Each research document should include:

- Objective and decision context
- Research parameters (`scope`, `depth`, `as_of`, rounds)
- Findings by track (codebase, docs, external, learnings)
- Dated evidence table (claim/source/date/checked/confidence)
- Contradictions and unresolved unknowns
- Recommendation and alternatives
- Revalidation plan/date
